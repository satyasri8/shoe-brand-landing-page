# Shoe Brand Landing Page

A modern and responsive landing page built with HTML, SCSS, and JavaScript. Designed to deliver an engaging user experience through clean layouts, responsive design principles, and smooth UI interactions.

## 🚀 Features

* Responsive design for mobile and desktop devices
* Interactive navigation menu
* Smooth animations and transitions
* Modern landing page UI
* Optimized layout using SCSS

## 🛠️ Technologies

* HTML5
* SCSS (Sass)
* JavaScript

## 🎯 Project Objective

This project was developed to strengthen frontend development skills, focusing on responsive web design, SCSS architecture, and user interface implementation.
